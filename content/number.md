Numbers in Lua
==============

```lua
25
12.2
1.12e10
3E-3
```

```lua
x = 12
print(type(x)) -- number
y = 13.3
print(type(y)) -- number
```

```lua
12 = 12.0 -- true
1e3 == 100 -- true
```

```lua
0xff -- 255
0x0.2 -- 125
```

integers
--------

```lua
xint = 12
math.type(xint) -- integer
```

floats
------

```lua
xflt = 12.0
math.type(xflt) -- float
```
